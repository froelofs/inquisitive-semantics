## Software and services

This website was built using the following software and services.

* [Jekyll](jekyllrb.com/): the static site generator.
* [Bootstrap](https://getbootstrap.com): our css framework.
* [BibDesk](https://bibdesk.sourceforge.io/) for managing BibTeX files
* [BibBase](https://bibbase.org/) for rendering the BibTeX bibliography.
* [GitHub Pages](https://pages.github.com/) for hosting the website.

  ouputs a value from an object, but uses a data file for the default values.
